Download — RU Soberano

Reator catalítico PFR · Windows · Lançamento 18 de Junho de 2026

Lançamento oficial: 18 de Junho de 2026. Abaixo: instruções de download e onde encontrar o instalador quando os releases forem publicados.

O RU Soberano é distribuído como executável Windows (RU_Soberano.exe). Inclui o console de engenharia (interface de controle PFR), simulador 3D (Panda3D) e kernel Langmuir-Hinshelwood. Requer Python no path ou runtime empacotado (build PyInstaller).

Versões:

  • FREE: experiência até o Buffet (sítio ativo); degustação do modelo cinético e do console.
  • SOVEREIGN: jornada completa, Cena Final e recursos exclusivos; ativação via license.key em config/.

Quando os releases forem publicados neste repositório, o instalador estará em:

Releases → Latest (GitHub)
